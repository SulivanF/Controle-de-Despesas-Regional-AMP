# Controlo de Despesas Regional Ampére

## Sistema Web de Gestão de Despesas

Este repositório contém o sistema de controle de despesas da Regional Ampére.

### 🌐 Acesse o Site

O site está disponível em: **https://sulivanf.github.io/Controle-de-Despesas-Regional-AMP/**

### 📋 Funcionalidades

- Dashboard com KPIs financeiros
- Registro de despesas (Almoço ou Danos/Outros)
- Gestão de técnicos por cidade
- Filtro por período (mês/ano)
- Modo claro/escuro
- Exportar relatórios em PDF
- Dados salvos automaticamente no navegador

### 🚀 Como Usar

1. Acesse o link do site acima
2. Navegue pela interface usando as abas:
   - **Dashboard**: Visualiza resumo de despesas
   - **Registar Despesa**: Adiciona novas despesas
   - **Cidades e Técnicos**: Gerencia técnicos

### 💾 Dados

Os dados são salvos automaticamente no navegador usando Local Storage. Não serão perdidos ao recarregar a página.

### 📄 Relatórios

Clique em "Exportar Relatório em PDF" para gerar um relatório completo.
